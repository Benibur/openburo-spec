## Goal Achievement

The milestone goal was: *A host app can call `obc.castIntent(intent, cb)` once and get a fully orchestrated file-picker / file-save flow — capability discovery, user selection, sandboxed iframe lifecycle, and PostMessage round-trip — with zero framework lock-in.*

This goal is **met**. The `OpenBuroClient` public class:
- Discovers capabilities via HTTP fetch (with WebSocket live updates)
- Resolves intents against capabilities using pure MIME matching
- Presents a chooser modal for multiple matches with full WCAG a11y baseline
- Opens a sandboxed cross-origin iframe for the selected capability
- Round-trips results via Penpal v7 (`connect` + `WindowMessenger`)
- Handles leak-free teardown via `createAbortContext()`
- Supports multiple concurrent instances and sessions
- Ships as tree-shakable ESM + CJS + UMD with correct TypeScript declarations
